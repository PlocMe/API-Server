{{=T('The game')}}


Participantes:

Ploc.Me é recomendado para crianças a partir de 8 anos, adultos e idosos. O número de jogadores é ilimitado.

Objetivo:

Somar mais pontos, sem limites. Para fazer pontos, você deve capturar as bolhas espalhadas nos principais espaços públicos.

A quantidade de pontos que você ganha é a soma dos pontos das bolha capturadas.

Haverá bolhas com bônus ou prêmios oferecidas pelos patrocinadores do jogo. É possível criar novos desafios e participar das campanhas dos anunciantes.

Regras:

Vale tudo! Capture quantas bolhas for possível.

Bolhas:

Para jogar Ploc.Me é necessário instalar o aplicativo no celular que tenha os recursos de GPS, acelerômetro e câmera. Ploc.Me é composto por 5 tipos de bolhas, sendo:

  • Bolha azul = 1 ponto
  • Bolha verde = 5 pontos
  • Bolha vermelha = 10 pontos
  • Bolha amarela = 20 pontos
  • Bolha bônus = Com bônus de patrocinadores
